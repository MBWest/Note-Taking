#  Common Ports

> ## **TCP**

| **PORT**   | **Description**   |
|:----------:|:-----------------:|
| `21` | FTP  |
| `22` | SSH |
| `23` | Telnet |
| `25` | SMTP |
| `53` | DNS |
| `80` | HHTP |
| `443` | HTTP |
| `110` | POP3 |
| `129 or 445` | SMB |
| `143` | IMAP |

---
---

> ## **UDP**

| **PORT**   | **Description**   |
|:----------:|:-----------------:|
| `53` | DNS  |
| `67, 68` | DCHP |
| `69` | TFTP |
| `161` | SNMP |
