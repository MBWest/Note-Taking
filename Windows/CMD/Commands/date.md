# date

> ## **Examples**

| **Command** | **Description** |
|-------------|-----------------|
| `date /t` | Display the date so you are not prompted to change it |
| `date 01/1/2017` | Display the current date and change it to 01/01/2017 |
| `date /t >> C:\date.txt` | Display the current date and output it to C:\date.txt |
