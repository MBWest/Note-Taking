# sort

- Sorts from A-Z by default

---

> ## **Examples**


| **Command** | **Description** |
|-------------|-----------------|
| `sort < cwo.txt` | Inputs cwo.txt into the sort command |
| `sort file1.txt` | Sort file1.txt from A-Z |
| `type file1.txt \| sort /r` | Display file1.txt sort from Z-A |
