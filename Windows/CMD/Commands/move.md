# move

- Moves a file from one location to the other

---

> ## **Examples**

| **Command** | **Description** |
|-------------|-----------------|
| `move CLI.txt C:\Exercise3a\USAF\MAJCOMs` | Move the C:\CLI.txt  file to the USAF\MAJCOMs folder |
| `move file1.txt C:\CWO` | Moves C:\file1.txt to C:\CWO folder |
