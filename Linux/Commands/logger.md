# logger

`logger [options] message`

## Examples

| **Command**   | **Description**   |
| --------------|-------------------|
| **Example** |
| `logger -p mail.info -t mailtest "Test."`


## Options

| **Command**   | **Description**   |
| --------------|-------------------|
| **Option** |
| `-p` | FACILITY.SEVERITY | 
| `-t` | TAG |
