# unset 

Unset values and attributes of variables and functions

---

> ## **Syntax**

- unset [-fv] name...

---

> ## **Example**

| **Command**   | **Description**   |
| --------------|-------------------|
| `unset -v VISUAL` | Unset VISUAL variable |
| `unset -f foo bar` |Unset the functions **foo** and **bar** |
