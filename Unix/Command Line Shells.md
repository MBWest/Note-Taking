# Comand Line Shells

## Popular Shells

1. **Bash** - `BASH`
2. **Korn Shell** - `ksh`
    - Acts like BASH
3. **C Shell** - `csh`
4. **TC Shell** - `tcsh`
    - Upgraded version of csh
5. **Z Shell** - `zsh`
    - Default Kali Linux Shell
